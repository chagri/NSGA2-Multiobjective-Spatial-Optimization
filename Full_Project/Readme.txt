The code runs using three files:
	Adj_List_file.gal : adj list for each cell
	Main_data_file : It has area, current landtype and all the possible landtypes in all the 5 cases (A-E), including the base case
	Targets_File: it has the target area for each land_type: allowed max and min limits

Requires: INSPYRED Python library as dependency

Execute: Final_NSGA2.py to run the code
